Technical Architecture

Built with Next.js 14 App Router, TypeScript, and Tailwind CSS following the TokenOS dark theme design system.

LAYER TECHNOLOGY PURPOSE
Framework Next.js 14 (App Router) Server components, routing, API routes
Language TypeScript (Strict) Type safety across all components
Styling Tailwind CSS TokenOS dark theme implementation
Real-time Socket.io Client WebSocket connection for live updates
State React Context Auth, WebSocket, and Toast providers
API Client Fetch with wrapper Type-safe API communication
