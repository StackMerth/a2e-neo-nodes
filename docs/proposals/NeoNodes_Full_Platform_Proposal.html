Bidirectional Market Routing

High Internal Demand: Node runners serve internal jobs at premium rates.
Low Internal Demand: Idle nodes automatically list on Akash/IO.net to earn from external demand.
Result: Node runners always earn — either from internal customers or external markets, calculated hourly.

Node deployment currently requires manual data center coordination. If Hydrahost or similar providers offer API-based provisioning, automated node deployment can be added as a future enhancement at no additional cost.

Why Node.js for Everything?

One language from backend to frontend to node agent means faster development, easier debugging, shared types and utilities, and no context switching. The entire team works in TypeScript. No Python, no Go, no fragmentation — just JavaScript everywhere.
